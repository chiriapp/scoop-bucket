# Chiri Scoop Bucket

[![Tests](https://github.com/chiriapp/scoop-bucket/actions/workflows/ci.yml/badge.svg)](https://github.com/chiriapp/scoop-bucket/actions/workflows/ci.yml)
[![Excavator](https://github.com/chiriapp/scoop-bucket/actions/workflows/excavator.yml/badge.svg)](https://github.com/chiriapp/scoop-bucket/actions/workflows/excavator.yml)

[Scoop](https://scoop.sh) bucket for [Chiri](https://github.com/chiriapp/chiri), a cross-platform CalDAV task management app.

## Install

```powershell
scoop bucket add chiri https://github.com/chiriapp/scoop-bucket
scoop install chiri/chiri
```
